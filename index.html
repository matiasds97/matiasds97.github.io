Habilidades

Unity

Mi opción principal a la hora de crear videojuegos 2D o aplicaciones interactivas. También lo utilizo para crear pequeños juegos en 3D y prototipos. Es el motor que utilicé para desarrollar Number Way para Android en 2017.

Unreal Engine 4

Actualmente me encuentro aprendiendo y descubriendo todas las posibilidades y herramientas que este motor puede brindar. Utilizo el sistema de Blueprints para programar juegos y los shaders. Mi primer videojuego en 3D, Cactus Hero, se encuentra en desarrollo en este motor.

Desarrollo en Android

Desarrollé una aplicación de Android llamada CreamApp donde los usuarios ganan puntos al comprar helado, los cuales pueden ser canjeados por más helado. El proyecto utiliza una comunicación con API mediante Retrofit, Servicios, sensores de Smartphone, tales como el acelerómetro, giroscopio y sensor de luz. Desarrollo principalmente en Java.

.NET Framework

Experiencia en desarrollo de Web Apps con arquitectura MVC en C#. También utilicé este framework para algunos proyectos de Angular y React. También use este framework para desarrollar pequeños proyectos de UniversalWindows Platform (UWP).

Blender

Usado principalmente para crear modelos 3D de mi juego Cactus Hero. También lo utilizo para crear algunos renders, usando el apartado de shaders y UV's.

Adobe Photoshop

Lo utilizo para crear ilustraciones e imagenes, principalmente para proyectos escolares y de la universidad. También lo empleo para crear ilustraciones e interfaces de usuario para videojuegos, aplicaciones y sitios web.

VEGAS Pro

Mi herramienta principal para crear proyectos con video. También lo utilizo para crear efectos de sonidos para mis proyectos.

Office 365

Mis principales aptitudes dentro de este paquete de software son aplicaciones Low-Code con Microsoft Powerapps y flujos de automatización con Microsoft PowerAutomate, integrado con SharePoint.
